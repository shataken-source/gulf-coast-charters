### 2. TRANSACTION FEES (High-Volume Revenue)

#### Booking Commission - 8%
- Captain charges customer $500 for trip
- Platform takes $40 (8%)
- Captain nets $460
- **First 3 bookings free** for new captains

**Why It Works:**
- Captain already paying credit card processing (2.9%)
- Platform provides value (marketing, booking system, customer base)
- Industry standard: 10-15%, we're lower

#### License & Document Processing - 5-10%
- User buys USCG license renewal ($200)
- Platform processes for $15-20 fee
- Covers admin time, verification, expediting

**Why It Works:**
- Convenience fee (saves hours of paperwork)
- Similar to DMV processing services
- Optional - users can still do it themselves

## üö´ WHAT NOT TO DO

### Never Do These:
1. ‚ùå Pop-ups that interrupt fishing reports
2. ‚ùå Full-screen ads before weather data
3. ‚ùå Limit basic weather/safety info (must be free)
4. ‚ùå Remove features from free tier after users rely on them
5. ‚ùå Ads that autoplay sound
6. ‚ùå Fake "limited time" offers that never end
7. ‚ùå Hide community features behind paywall
8. ‚ùå Require payment to report hazards/safety issues

### Keep These Free Forever:
- Basic buoy data (nearest buoy)
- Current weather conditions
- Hazard warnings
- Emergency info
- Coast Guard contacts
- Basic tide data
- Community access (read)
- Essential safety features
